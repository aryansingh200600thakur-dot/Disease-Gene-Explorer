\# Disease Gene Explorer



A bioinformatics project that identifies genes associated with diseases using NCBI databases.



\## Features



\- Search diseases

\- Retrieve associated genes

\- View gene descriptions

\- View chromosome locations

\- Access NCBI source links



\## Technologies Used



\- Python

\- FastAPI

\- NCBI API

\- Git

\- GitHub



\## Run the Project



```bash

cd backend

python -m uvicorn app.main:app --reload

```



Open:



http://127.0.0.1:8000/docs
